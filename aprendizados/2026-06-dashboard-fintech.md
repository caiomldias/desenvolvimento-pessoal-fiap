# Aprendizado: Dashboard Fintech Fluxa

## Contexto

Durante o desenvolvimento do projeto Fluxa, trabalhei na criacao de uma tela de dashboard financeiro para uma entrega academica da FIAP.

## O que aprendi

- Separar estrutura HTML e estilo CSS
- Criar layout responsivo para diferentes resolucoes
- Usar paleta visual consistente com uma identidade de produto
- Preparar um projeto para funcionar 100% offline
- Organizar arquivos para entrega em ZIP
- Publicar um projeto no GitHub usando Git e GitHub CLI

## Decisoes importantes

- Remover dependencias externas para evitar problemas na avaliacao
- Manter fontes locais dentro da pasta do projeto
- Usar SVG inline para evitar depender de bibliotecas de icones
- Criar um README de perfil para melhorar minha apresentacao no GitHub

## Proximos pontos para evoluir

- Aprender JavaScript para tornar dashboards interativos
- Melhorar acessibilidade em componentes de interface
- Criar documentacao mais completa para cada projeto
- Praticar commits pequenos e bem descritos
