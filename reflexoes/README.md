# Reflexoes

Espaco para registrar minha evolucao pessoal como estudante e futuro profissional de tecnologia.

## Perguntas uteis

- O que eu aprendi nesta etapa?
- Qual foi minha maior dificuldade?
- O que eu faria diferente em uma proxima entrega?
- Qual habilidade tecnica preciso reforcar?
- Qual habito de estudo funcionou melhor?
